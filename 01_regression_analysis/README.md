# Regression Analysis on Italian Household Expenditure

## Project Overview

This project analyzes the determinants of average monthly household expenditure in Italy using multiple linear regression models.  

The objective is to understand how socio-economic and demographic factors influence household consumption patterns using official ISTAT microdata.

---

## Dataset

The analysis is based on publicly available ISTAT data on Italian household consumption expenditures.

---

## Variables

### Dependent Variable
- Average monthly household expenditure (in euros)

### Independent Variables
- Geographic area of residence (North, Centre, South, Islands)
- Household composition
- Employment status of the reference person
- Type of expenditure

---

## Methodology

The analysis follows a classical econometric approach:

- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Encoding of categorical variables
- Multiple Linear Regression
- Model diagnostics and validation

---

## Tools & Technologies

- R
- tidyverse
- ggplot2
- statistical modeling (lm function)
- data visualization

---

## Key Objectives

- Identify key drivers of household expenditure
- Quantify the impact of socio-economic factors
- Interpret regression coefficients in an economic context
- Evaluate model performance and assumptions

---

## Results 

- Significant predictors of household expenditure
- Regional differences in consumption patterns
- Effect of employment status and household structure
